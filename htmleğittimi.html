  • HTML (HyperText Markup Language),
    Tim Berners-Lee tarafından 1990 yılında World Wide Web'in temel bileşeni olarak geliştirilmeye başlandı.
    İlk HTML sürümü olan HTML 1.0, 1991 yılında yayımlandı.
    Bu sürüm, temel yapıyı tanımlayan etiketler ve özellikler içeriyordu.
  • >HTML'nin ilk sürümleri basit ve sınırlı özelliklere sahipti.
    Ancak webin hızla büyümesi ve gelişmesiyle birlikte daha karmaşık ve etkileşimli web sayfaları ihtiyacı ortaya çıktı.
    Bu nedenle HTML, zamanla yeni özellikler ve sürümlerle gelişti.
  • İşte HTML'nin tarihçesi ve ana sürümleri:
  • >HTML 2.0: 1995 yılında yayımlanan HTML 2.0, ilk resmi HTML spesifikasyonu olarak kabul edildi.
    Bu sürüm, tablolar, form alanları, resimler gibi yeni özellikleri tanıttı.
  • HTML 3.2: 1997 yılında yayımlanan HTML 3.2, daha fazla etiket ve özellik ekledi.
    CSS (Cascading Style Sheets) desteği de HTML 3.2 ile başladı.
  • HTML 4.01: 1999 yılında yayımlanan HTML 4.01, birçok gelişmiş özellik ve stil seçenekleri sunarak web tasarımına daha fazla esneklik sağladı.
    Ayrıca, HTML 4.01 Strict ve HTML
  • XHTML (Extensible HyperText Markup Language): XHTML, XML (Extensible Markup Language) tabanlı bir HTML versiyonudur.
    XHTML, daha katı bir sözdizimine sahiptir ve XML standartlarını takip eder.
    XHTML 1.0, 2000 yılında yayımlanan bir standarttır.
  • HTML5, daha zengin ve etkileşimli web deneyimleri sağlamak için yeni öğeler, medya desteği, yerel depolama, form kontrolleri, canvas gibi bir dizi yenilikçi özellik ve API sunar.
